# Quick Log Plus

Insert colored `console.log` statements fast in JavaScript and TypeScript.

![](https://github.com/tamas-avalishvali/quickLog/raw/main/stringlog.gif)
## Features

- Replaces a keyword with a styled `console.log`.
- Adds color to console output.
- Works with strings and variables.
- Runs on space trigger.
- Speeds up debugging.

:::note[]
## How it works

Type a value and then a keyword with a space:
Write the target word first.
Press the required console key next.
Then **add a space** at the end to trigger the log.
:::warning[]
Make sure to include a **space** at the end of the log, and do not press any key other than the space.
:::
